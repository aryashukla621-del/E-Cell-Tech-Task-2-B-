Number Guessing Game
This is my submission for Task 2 (B) in E-Cell tech recruitment.

What it does
It's a simple number guessing game where you guess a number between 1 and 100.

Features
Tells if your guess is too high or too low
Counts how many tries you took
Shows your previous guesses
Option to restart the game
Tech Used
HTML
CSS
JavaScript
How to Run
Open index.html in your browser and start playing.

What I Learned
Handling user input and updating the page dynamically
